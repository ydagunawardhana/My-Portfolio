## ✨ Features & Functions

This portfolio is engineered to provide a premium user experience with dynamic interactions:

- **Fully Responsive Layout**: Built mobile-first, ensuring perfect rendering across all device sizes.
- **Dark/Light Mode Theme**: Seamless toggling between light and dark themes using `next-themes`.
- **Framer Motion Animations**:
  - **Scroll Reveals**: Sections fade and slide into view as the user scrolls.
  - **Staggered Timelines**: Beautifully cascading animations for the Certifications timeline and Project cards.
  - **Animated Skill Progress Bars**: Dynamic progress trackers that visually count up from 0% when entering the viewport.
- **Smooth Navigation**: Custom smooth scrolling anchors integrated directly into the React routing.
- **Interactive Data Display**: Complex UI components for rendering project portfolios, tech stacks, and professional experiences.

## 🛠 Tech Stack

The architecture relies on the following cutting-edge tools:

- **[Next.js 13+](https://nextjs.org)** - React framework utilizing Server Components for optimized performance.
- **[TypeScript](https://typescriptlang.org)** - Static type checker for end-to-end type safety.
- **[Tailwind CSS](https://tailwindcss.com)** - Utility-first CSS framework for rapid and highly customizable UI development.
- **[Framer Motion](https://www.framer.com/motion/)** - Production-ready animation library for React, powering all the scroll and stagger effects.
- **[Radix UI](https://www.radix-ui.com/)** - Unstyled, accessible primitives for building robust interactive components.
- **[Lucide Icons](https://lucide.dev)** - Clean, crisp, and beautifully simple SVG icons.

## 🚀 Running Locally

To run this project locally on your machine:

```bash
# Clone the repository
$ git clone https://github.com/yashandinusha/yashan.dev.git

# Navigate into the project directory
$ cd yashan.dev

# Install dependencies
$ npm install

# Start the development server
$ npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
